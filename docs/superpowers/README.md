# Historical planning artifacts

The plans and specifications in this directory were created during Janus development in June and July 2026. They are preserved to explain design decisions, but they are not a current implementation tracker and unchecked plan boxes do not necessarily mean the corresponding feature is missing.

This directory is excluded from the published documentation site in `mkdocs.yml`.

- `plans/` contains implementation plans written before or during development.
- `specs/` contains the design specifications those implementations followed.

For current behavior, see [`../architecture.md`](../architecture.md) and the source code. For active work, see [`../../todo.md`](../../todo.md). `ISSUES.md` and `STRIX-AUDIT.md` are dated point-in-time reports whose status notes identify known stale findings.
